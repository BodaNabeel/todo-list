# todo-list-webapp
# Brief Introduction: 
To-do list is a CRUD webapp which has all the feature of a to-do list, in it a user can keep track of
its tasks to help maintain it's productivity
# Functionalitites:
* Adding a task
* Marking a task as done
* Editing a task
* Delete a task
# Technology used:
* HTML5
* CSS3
* Vanilla Javascript
